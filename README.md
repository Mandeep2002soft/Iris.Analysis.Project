# Iris Flower Analysis

## What is this?
I looked at 150 iris flowers. There are 3 types: Setosa, Versicolor, Virginica.

## What I found
- Setosa has smallest petals
- Virginica has largest petals  
- Petal length tells you which flower it is

- Python
- Pandas
- Matplotlib
